# Android Automation Project

## Overview
This project is an automated testing framework for Android applications using WebdriverIO and TypeScript. The goal is to provide a comprehensive suite of automated tests for continuous integration and delivery.

## Features
- Automated UI testing for Android applications
- Written in TypeScript for type safety
- Easily extensible framework
- Detailed reporting

## Requirements
- Node.js
- NPM/Yarn
- Android emulator or device

## Installation
1. Clone the repository:
   ```bash
   git clone https://github.com/bishal98gupta/WDIO_project.git
   cd WDIO_project
   ```
2. Install the dependencies:
   ```bash
   npm install
   ```

## Configuration
Configure WebdriverIO by editing the `wdio.conf.ts` file. Set the desired capabilities for your Android device.

## Running Tests
To execute the tests, run the following command:
```bash
npm run test
```
